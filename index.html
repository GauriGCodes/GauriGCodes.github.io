My Projects

NYSPHA Advocacy Tools

The NYSPHA Advocacy Tool is a website that allows NYSPHA members to contact their elected officials about public health issues in a few easy steps. Members can select the issue they wish to take action on, provide their contact information and the email addresses of the elected officials they wish to contact, and add a personal message to their email. Once the form is complete, members can click the "Send Message" button to send their email.

SpringBoot Mailer API

Spring Boot Mailer API that allows users to send automated email updates and confirmations to any email address, triggered by events or cron jobs. It supports plain text, HTML, and MIME emails, attachments, dynamic content, and email delivery and opens tracking. It can be used to send automated emails for a variety of purposes, such as welcome emails, order confirmations, shipping notifications, weekly newsletters, and subscription reminders.

Log Search Utility

Full Stack Application (Log Search Utility) to enhance log analysis efficiency by 70%. It is built with React, Spring Boot, Spring Security LDAP, and Apache Lucene, and it enables easy log searching and downloading for BAs and Developers. The Log Search Utility provides significant benefits, including improved log analysis efficiency, increased productivity, enhanced security, and scalability.
